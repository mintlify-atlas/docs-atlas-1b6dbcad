  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with wt in minutes
  </p>
  
  <Steps>
    <Step title="Install wt">
      Clone the repository and run the installation script:
      
      ```bash
      cd ~
      git clone https://github.com/uglyus-daniel/wt.git
      cd wt
      chmod +x wt
      ./wt install
      source ~/.zshrc
      ```
      
      The `wt install` command handles script installation, zsh function registration, and automatic fzf setup.
    </Step>
    
    <Step title="Create your first worktree">
      Navigate to any git repository and create a new worktree:
      
      ```bash
      cd ~/your-project
      wt new feature/user-auth dev
      ```
      
      This creates a new worktree based on the `dev` branch, automatically:
      - Creates the branch if it doesn't exist
      - Copies your `.env` file
      - Installs dependencies with the correct package manager
      - Changes directory to the new worktree
    </Step>
    
    <Step title="Start development">
      Run your development server in the new worktree:
      
      ```bash
      wt dev
      ```
      
      The `dev` command automatically detects your package manager (bun, pnpm, yarn, or npm) and runs the appropriate dev command.
    </Step>
  </Steps>
</div>
